# Sistema de Cadastro de Livros

Projeto desenvolvido em HTML, CSS e JavaScript.

## Funcionalidades

- Login de usuário
- Cadastro de livros
- Listagem dos livros
- Busca por título
- Edição dos dados
- Exclusão dos registros

## Tecnologias

- HTML
- CSS
- JavaScript
- LocalStorage
